# Feature: Goal Pack — IELTS Writing

## Intent
Make steady gains to reach ≥6.5 writing by exam date **2025-11-25**.

## Acceptance Criteria
1. Quick-create a **daily IELTS session** with template:
   - Prompt → Outline 5’ → Draft 25’ → Rubric 10’ → Error log → Next fix.
2. Timer presets: default 3 Pomodoro cycles (25/5) within 09:00–10:30.
3. Weekly mock generator (Task1 + Task2) with review slot.
4. Error log with taggable patterns and “assigned fix” links.
5. Trend chart of rubric subscores week-over-week.

## Notes
- Sessions tag as **@deep** by default; scheduled in peak window.
