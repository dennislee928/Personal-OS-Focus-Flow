# Global Acceptance Criteria

## A. Rituals
- Daily Start completes ≤ 10 minutes with the required steps performed.
- Weekly Review completes ≤ 40 minutes and schedules at least 3 deep blocks.

## B. Focus & WIP
- During the first scheduled deep block of the day, WIP in Doing ≤ 1.
- Any WIP breach logs an event and shows an inline explanation.

## C. Scheduling & Peak Window
- Default placement for first deep block falls within 09:00–12:00 unless user overrides.

## D. Persistence
- All ritual outcomes, board states, and logs are persisted and recoverable after restart.

## E. Packaging (Release)
- macOS: signed + notarized `.dmg` builds produced on macOS runner.
- Windows: signed NSIS `.exe` builds produced on Windows runner.
